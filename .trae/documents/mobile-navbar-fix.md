# 首页移动端导航栏修复计划

## 问题描述

1. **汉堡图标颜色问题**：移动端汉堡图标为白色（#f5f5f7），在白色导航栏背景下不可见
2. **下拉菜单无法收回**：移动端菜单展开后无法关闭

## 修复方案

### 问题 1：汉堡图标颜色

**当前样式**：
```css
.hamburger-line {
    background: #f5f5f7;  /* 白色，在白色背景下不可见 */
}
```

**修复为**：
```css
.hamburger-line {
    background: #333;  /* 深色，在白色背景下可见 */
}
```

### 问题 2：菜单无法收回

**原因分析**：
- 移动端菜单展开后，缺少关闭机制
- 需要添加点击菜单项后自动关闭功能
- 需要添加点击遮罩层关闭功能

**修复方案**：
1. 修改 JavaScript 代码，添加菜单项点击后关闭菜单
2. 添加点击菜单外部区域关闭菜单
3. 确保汉堡按钮点击可以切换菜单状态

## 实施步骤

### Step 1: 修改汉堡图标颜色
将 `.hamburger-line` 的 `background` 从 `#f5f5f7` 改为 `#333`

### Step 2: 修复 JavaScript 菜单逻辑
找到导航栏 JavaScript 代码，添加：
- 菜单项点击后自动关闭菜单
- 点击菜单外部区域关闭菜单
- 确保汉堡按钮正确切换菜单状态

### Step 3: 验证修复效果
- 移动端汉堡图标可见
- 点击汉堡图标菜单展开
- 再次点击汉堡图标菜单收起
- 点击菜单项后菜单自动收起
