# 优化团队 PIN 码显示效果计划

## 任务概述
优化 `/source/ztsm/index.html` 页面底部团队 PIN 码卡片区域的视觉效果和性能

## 当前实现分析
- 使用毛玻璃卡片 + 动态光斑背景
- 三个 blob 使用 `filter: blur(60px)` 可能有性能问题
- 动画使用 `animation` 属性

## 详细实施步骤

### 步骤 1：优化背景光斑性能
- 移除 `filter: blur(60px)` 使用纯色半透明背景替代
- 减少 blob 数量从 3 个到 2 个
- 简化动画，仅使用 CSS transform（GPU 加速）
- 添加 `will-change: transform` 提示浏览器优化

### 步骤 2：增强卡片视觉效果
- 优化毛玻璃效果参数
- 添加渐变边框效果
- 增加悬停时的光晕效果
- 优化 PIN 码字体样式

### 步骤 3：改进交互体验
- 添加复制成功动画反馈
- 优化按钮过渡效果
- 添加 focus 状态样式

### 步骤 4：优化响应式设计
- 确保移动端显示正常
- 调整间距和尺寸适配小屏幕

## 预期效果
- 页面加载性能提升
- 视觉效果更加精致
- 交互反馈更加流畅
