# 导航栏图标文字间距调整规范

## Why
当前首页顶部导航栏的选项图标与文字之间几乎没有间距，看起来贴在一起，影响视觉效果和可读性。

## What Changes
- 调整 `.nav-link` 的 CSS 样式，增加图标与文字之间的间距
- 使用 `gap` 属性实现一致的间距

## Impact
- 受影响文件：`index.html`（导航栏 CSS 样式）
- 用户体验：更舒适的导航项阅读体验

## ADDED Requirements
### Requirement: 导航项图标与文字间距
系统 SHALL 确保导航项的图标与文字之间有适当的间距

#### Scenario: 桌面端导航
- **WHEN** 用户查看导航栏
- **THEN** 每个导航项的图标与文字之间有 8px 的间距

#### Scenario: 移动端导航
- **WHEN** 用户在移动端查看导航栏
- **THEN** 每个导航项的图标与文字之间有 6px 的间距
